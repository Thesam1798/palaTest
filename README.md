# PalaTest (Forge 1.7.10 - 10.13.4.1614)

## Building and setting up

Run the following commands in the root directory:

```shell
./gradlew clean build
```

## Run Client

Run the following commands in the root directory:

```shell
./gradlew clean runClient
```

## Run Server

Run the following commands in the root directory:

```shell
./gradlew clean runServer
```
